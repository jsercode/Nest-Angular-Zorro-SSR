## 开始使用

基于Nest + AngularCLI + ZORRO 搭建服务端渲染(SSR)方案

### 安装依赖

- `yarn install` 或 `npm i`

### 客户端渲染

- `npm start`

### 服务器端渲染

- `npm run dev:ssr`.

### 生产环境

\*`npm run build:ssr && npm run serve:ssr`

- 启动服务为: `http://localhost:4000`.

\*`npm run prerender`
